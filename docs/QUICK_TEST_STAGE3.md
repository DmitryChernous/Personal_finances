# Быстрая проверка Этапов 3-4

## После успешного push (`npm run push`):

1. **Откройте Google Таблицу** и обновите страницу (F5)

2. **Запустите Setup**: **Personal finances → Setup (создать листы)**
   - Проверьте, что созданы листы `Отчеты` и `Дашборд`

3. **Сгенерируйте тестовые данные** (автоматически!):
   - **Personal finances → Заполнить тестовыми данными**
   - Выберите "Да" для очистки существующих данных (если нужно)
   - Будет создано:
     - 5 счетов (Наличные, Карта Сбер, Карта Тинькофф, Вклад, Доллары)
     - 15 категорий (доходы и расходы с подкатегориями)
     - ~60 транзакций за последние 3 месяца (доходы, расходы, переводы)

4. **Проверьте лист `Отчеты`**:
   - Секция "Сводка по периодам" — должны быть суммы
   - Секция "Топ расходов по категориям" — таблица с категориями
   - Секция "Остатки по счетам" — остатки для каждого счёта

5. **Проверьте лист `Дашборд`**:
   - KPI карточки: Доходы, Расходы, Итого, Средний расход в день
   - График "Расходы по категориям" (pie chart)

6. **Проверьте обновление**: добавьте новую транзакцию → суммы должны обновиться автоматически

Если всё работает — можно продолжать разработку!
