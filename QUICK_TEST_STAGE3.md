# Быстрая проверка Этапа 3

## После успешного push (`npm run push`):

1. **Откройте Google Таблицу** и обновите страницу (F5)

2. **Запустите Setup**: **Personal finances → Setup (создать листы)**
   - Проверьте, что создан лист `Отчеты`

3. **Добавьте тестовые данные**:
   - В `Счета`: "Наличные", "Карта"
   - В `Категории`: "Еда", "Транспорт", "Зарплата"
   - В `Транзакции`: несколько доходов и расходов за текущий месяц

4. **Проверьте лист `Отчеты`**:
   - Секция "Сводка по периодам" — должны быть суммы
   - Секция "Топ расходов по категориям" — таблица с категориями
   - Секция "Остатки по счетам" — остатки для каждого счёта

5. **Проверьте обновление**: добавьте новую транзакцию → суммы должны обновиться автоматически

Если всё работает — можно переходить к Этапу 4!
